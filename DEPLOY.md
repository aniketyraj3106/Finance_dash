# Finance Dashboard - Quick Deployment Guide

## 🚀 VERCEL (Recommended - Fastest)

**Time: 2 minutes**

1. Go to https://vercel.com/new
2. Click "GitHub" → Sign in
3. Search "Finance_dash" → Import
4. Click "Deploy"
5. Done! 🎉

Live URL: https://finance-dash.vercel.app

---

## 🚀 NETLIFY (Alternative)

**Time: 3 minutes**

1. Go to https://netlify.com
2. Click "Add new site" → Select repository
3. Choose "aniketyraj3106/Finance_dash"
4. Build settings auto-fill
5. Click "Deploy site"

Live URL: https://your-site.netlify.app

---
