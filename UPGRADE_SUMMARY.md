## Quick Troubleshooting

**Port already in use?**
```bash
npm run dev -- --port 3000
```

**npm install stuck?**
```bash
npm cache clean --force
npm install
```

**Want to reset data?**
1. Open DevTools (F12)
2. Storage → localStorage
3. Delete "finance-store"
4. Refresh page
