## 🆘 QUICK TROUBLESHOOTING

**Issue:** Build fails
**Solution:** `npm install && npm run build` (test locally)

**Issue:** Blank page
**Solution:** Check browser console (F12), verify index.html

**Issue:** Old version shows
**Solution:** Hard refresh (Ctrl+Shift+R), clear cache

**Issue:** Data missing
**Solution:** Check localStorage in DevTools → Application

**Issue:** Mobile not responsive
**Solution:** Test on actual device, check viewport meta tag
